Overview
Rythmix is a conceptual AI-powered lyric generator designed to help music artists create new songs ideas.
This project focuses on the Homepage and Navigation system, which guide users through different parts of the app after login

Compnenets
1) Homepage.java
   - Displays the main homepage after user login
   - Includes options for navigating to:
     - Generate New Lyrics
     - View Saved lyrics
     - Profile Page
     - Settings
     - Logout
2) Navigation.java
   - Handles the logic behind user selections.
   - Ensures smooth transitions between different app features.
   - Designed for scalability and future features integration.

How it works
After running `HomePage.java`:
1) The user sees the Rhytmix welcome screen.
2) A menu appears with option (1-5).
3) User input is sent to the `Navigation` class, which responds accordingly .
4) The loop continues until the user selects logout.
